## 使用方法

必須安裝 node 及擁有檔案 777 權限

1. 打開 cmd 開啟本資料夾
2. 輸入 `node index.js <filename.js>`
3. 完成

*只支援 UTF-8 文件 其餘僅寫入不支援文字*
